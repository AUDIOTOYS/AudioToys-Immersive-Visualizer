# AudioToys Immersive Speaker Visualizer

Standalone interactive speaker layout visualizer for a 42-speaker immersive audio room.

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Open the local URL shown in terminal.

## Build standalone HTML

```bash
npm run build
```

The standalone file will be here:

```txt
dist/index.html
```

You can rename it:

```txt
AudioToys-Immersive-Speaker-Visualizer.html
```

and open it directly in a browser.

## Notes

The final HTML is generated as a single-file bundle using Vite and `vite-plugin-singlefile`.
